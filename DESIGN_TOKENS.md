# Design Tokens — shadcn/ui (Maia, Neutral)

> Формат цветов: `oklch(lightness chroma hue)` — можно конвертировать из HEX на https://oklch.com
>
> **Конфигуратор тем:** https://ui.shadcn.com/themes

---

## Общие

| Токен         | Назначение                                | Значение                       |
| ------------- | ----------------------------------------- | ------------------------------ |
| `--radius`    | Базовое скругление углов (0 = квадратные) | `0`                            |
| `--font-sans` | Основной шрифт                            | `"Geist Variable", sans-serif` |
